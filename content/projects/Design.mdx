#2 Brand a Band

Ik heb met een Metodische aanpak in een groep de branding gecrëeerd voor een beginnende artiest. Hieronder valt een

Ook heb ik geleerd om samen als groep aan een brandguide te houden zodat we samen consistent dezelfde producten leverden. En dit natuurlijk allemaal met baat van de klant ging.

#2 AutohurenCuracao

#3 Wat heb ik gedaan 
Ik heb voor een Autoverhuurbedrijf een website gemaakt samen met een groep. Vooraf aan de website te maken heb ik de branding verzorgd zodat deze ook up to date was. 
Hierbij heb ik in een relatief korte periode gezorgd voor een hernieuwd logo en een vaste huisstijl. Dit heb ik Metodisch gedaan door te kijken naar wat de klant wilde uitstralen 
dit waren mensen die graag over wilde komen als een lokaal verhuurbedrijf dus heb ik ook de vlag van curacao als inspiratie genomen. 
[Curacao onderzoek](/Curacaouitkomsten.pdf)
#3 Wat heb ik hiervan geleerd 

Ik heb geleerd om snel en efficient een branding en logo te maken voor een klant. Dit was met beperkte feedback ( alleen maar van mijn groepje ) 
Omdat dit niet echt mogelijk was om aan de klant te vragen. Ook heb ik geleerd om dit parralel te maken met de rest van de website design zodat ik geen achterstand op zou lopen op de rest.

Wat is nou de inzicht die ik heb gekregen na dit semester ? 
ik heb geleerd dat een opdrachtgever / klant niet altijd gelijk heeft. En dat je hier best ideeën op mag inbrengen, als een klant niet helemaal weet wat ze nog zelf wilt ( eline ) kan je er voor kiezen je eigen creativiteit
z'n gang te laten gaan. Ook kan je de opdracht zelf een beetje aanpassen dat het je echt meer ligt. En dat je er meer mee kan (AutohurenCuracao) De opdracht daar bijvoorbeeld was dat we alleen een website re-design maakte
maar ik vond het ook erg belangrijk dat het logo, en het kleurenpallet hierop werd aangepast. Ook heb ik het inzicht gekregen over welke tools het meest efficient werken. Zo heb ik in een korte tijd leren werken met de Adobe toolset. 
En dit niet alleen op een laptop, maar ook op een iPad zodat ik extra gedetaileerde schertsen kon maken.
