Wat heb ik hiervan geleerd

Ik heb vooral geleerd hoe ik samen moet werken als een groep om verschillende itteraties samen te laten komen, ik had hier deze sprint erg veel moeite mee omdat ik dingen anders wilde designen dan mijn groepsgenoten. 
Ook heb ik geleerd hoe ik een idee kan omzetten naar een design, wel heb ik op het begin de fout gemaakt door dit puur te doen op mijn eigen subjectieve mening. Wat niet de bedoeling is. Waardoor ik later mijn designs heb moeten aanpassen 
wel heb ik daardoor ervan geleerd hoe ik die cmd methodes juist kan toepassen. En hoe ik dit ook efficient doe. Ook weet ik nu hoe ik uberhaupt een designfase goed kan doorlopen, met usertests, wireframes en prototype testing. 
Dat is ook gelijk wat ik erg fijn vond aan mijn groepje, ze wisten goed hoe je dit met wat structuur kon aanpakken waar ik ook veel van heb geleerd. 

De inzicht die ik heb grekegen van dit project, is dat het wel essentieel is om met je stakeholder contact te houden. Aangezien dit naar mijn mening niet genoeg gebeurde tijdens dit project, waardoor je moest commiten aan bepaalde ideeen. zonder concrete feedback te hebben. 
Het was idealer geweest als ik tussentijds nog met happyhorizon had kunnen spreken om feedback op te vragen.
